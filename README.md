# React-Portfolio

## Deploy
- NPM START (App)
  - CD into 'portfolio' folder
  - NPM START   
  - Runs on http://localhost:3000
- Sanity Start (Studio to add/remove projects/blogs)
  - Runs on http://localhost:3333
